## 安装 ##

1. 安装 acrylamid：

    ```
    pip install acrylamid
    ```

1. 安装 markdown：

    ```
    pip install markdown
    ```

1. 安装 translitcodec：

    ```
    pip install translitcodec
    ```
    如果不装的话，会出现警告：

    ```
    warn  no 'translitcodec' found, using NFKD algorithm
    ```

1. 安装 jinja2：

    ```
    pip install jinja2
    ```

## 初始化 ##

其实就是创建blog，用以下的命令：

```
acrylamid init /path/to/blog/
```

创建成功后的目录中有这些内容：

1. `conf.py`

    配置文件

1. content

    所有写的文章会自动放到这里，保存成纯文本格式

1. theme

    主题，相当于blog的模板

## 设置 ##

用文本编辑器打开 `conf.py`，设置语言为英语：

```
# -*- encoding: utf-8 -*-
# This is your configuration file.  Please write valid python!
# See http://posativ.org/acrylamid/conf.py.html

LANG = 'en' # 本来可能没有这一项，把它加在这里就行了
```

如果不设置语言为英语的话，等写完blog开始编译的时候，就会出现如下警告：

```
warn  no hyph-definition found for 'zh'
```

意思是无法为中文使用连字符，所以你如果不想把语言设置成英语，就在 `conf.py` 中关掉连字符的设定也可以避免看到这个警告：

```
FILTERS = ['markdown+codehilite(css_class=highlight)', 'hyphenate', 'h1']
```

改成：

```
FILTERS = ['markdown+codehilite(css_class=highlight)', 'h1']
```

## 写新文章 ##

使用这个命令：

```
acrylamid new 'your entry title'
```

尽量不要用中文，否则会无法生成友好的slug。

acrylamid 会自动帮你创建好一个文本文件，并调用你的默认文本编辑器打开，就可以开始写了，写完并保存后，运行下面的命令来编译它（就是生成静态页面）。

```
acrylamid co
```

生成静态页面后，可以用如下命令来查看效果：

```
acrylamid view
```

默认会调用 8000 端口打开，如果你的 8000 端口被占用了，可以指定其它端口：

```
acrylamid view -p 1234
```

用浏览器打开 `127.0.0.1:8000` 或是将 `8000` 换成你指定的端口来访问就可以了。
